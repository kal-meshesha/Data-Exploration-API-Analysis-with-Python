# Data Exploration with Python
## Exploratory Data Analysis | API Integration | Business Insights
### Executive Summary

A New York coffee shop chain needed insight into sales performance, product profitability, and customer behavior. Additionally, I practiced extracting environmental data using an external API.

Using Python, I cleaned and explored 50,000+ transaction records to analyze product margins, customer spending patterns, and profit drivers. I also demonstrated API integration by retrieving air quality data using automated requests.

The analysis identified high-margin products, valuable customer segments, and sales trends that support data-driven marketing and inventory decisions.

### Business Problem
Coffee Shop Analysis

The coffee shop owner wanted to understand:

Which products generate the highest margins?

Which customer groups spend the most?

Which products drive the most profit?

Which customer segment should be targeted for marketing?

Raw transactional data needed cleaning and structured analysis to produce meaningful business recommendations.
